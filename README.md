AI Store Intelligence System

## Overview

An AI-powered retail analytics platform built using CCTV footage, computer vision, and real-time analytics.

## Features

* Person Detection using YOLOv8
* Multi-Object Tracking
* Unique Visitor Counting
* Event Generation
* Heatmap Analytics
* Occupancy Monitoring
* FastAPI APIs
* Streamlit Dashboard

## Tech Stack

* Python
* YOLOv8
* OpenCV
* FastAPI
* Streamlit
* Pandas

## Business Value

* Analyze customer movement
* Monitor store occupancy
* Detect high footfall periods
* Improve product placement decisions
* Support operational insights

## Future Improvements

* Staff vs Customer Classification
* Conversion Analytics
* Product Zone Intelligence
* Real-time Alerts
